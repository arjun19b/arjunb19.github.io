Spotify Statistics Android App

Context: Full‑stack Android application using Spotify Web API + Firebase auth to deliver listening insights and mini‑games.

Overview

  • Displays top tracks, artists, and albums via Spotify API endpoints.
  • Implements authentication and account features with Firebase.
  • Includes mini‑games (Wordle, memory, brick breaker) seeded by user’s music data.

Architecture

  • Android (Java/XML) client with MVVM patterns.
  • Auth with Firebase; token handling for Spotify requests.
